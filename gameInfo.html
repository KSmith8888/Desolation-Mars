Controls

Controls with keyboard or mouse. Either can be used interchangably at any time.

  • Control movement with the arrow keys, the "w", "a", "s", "d" keys or by clicking the space you want to move to.
  • Access the game menu with the "p" key or by clicking the menu icon in the top left corner.
  • Within the item menu, focus or hover on an item to view the description. Click or press enter on an item to use it.

System Requirements/Info

  • Currently supports desktop (mobile version planned).
  • Tested for Chrome, Firefox, and Edge browsers.
  • Save data is stored in the browser. If you clear your browser history, your game progress will be lost.
  • This game requires that javascript is turned on to process the canvas.
